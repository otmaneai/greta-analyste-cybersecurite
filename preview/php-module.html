Architecture d'un site PHP + MariaDB

Module 1 54 min Développement PHP

Lire la chaîne navigateur vers Apache puis PHP et MariaDB à travers le site parcinfo, avec les contraintes de déploiement réellement rencontrées.

Ce module rend visible la logique de paramètres imposés par le code, les permissions Linux et la cohérence serveur vers application puis base.

Ouvrir la première leçon Retour au parcours

Prérequis

  • Comprendre le rôle d'un serveur web.
  • Lire une base MariaDB dédiée et un utilisateur SQL dédié.
  • Suivre un déploiement Linux simple.

Learning objectives

Ce module doit rendre la progression visible.

  • Identifier les briques techniques qui composent une application PHP dynamique.
  • Relier paramètres d'application, permissions Linux et base de données.
  • Préparer la lecture de WordPress comme application PHP complète.
