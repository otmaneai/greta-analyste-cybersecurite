Story

Le parcours où le langage devient produit.

La narration PHP n'est pas isolée. Elle relie la logique serveur, la base MariaDB, la réécriture d'URL, les formulaires, l'échappement et le contexte WordPress pour assumer la stack étudiée comme identité du produit.

Applied stack

PHP Apache2 MariaDB WordPress Tailwind CSS Alpine.js

Ce parcours est visible à la fois dans les contenus et dans le code qui structure la plateforme.
